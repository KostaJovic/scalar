Setup Instructions
1. **Create a** `.env` **file** in the project root and add your MongoDB credentials:
```
MONGODB_URI="mongodb+srv://<DB-USER>:<DB-PASSWORD>@db1.ezjzf5r.mongodb.net/"
```
Replace `<DB-USER>` and `<DB-PASSWORD>` with your actual MongoDB Atlas credentials.

2. **Install dependencies**:
```
npm i
```
3. **Start the development server**:
```
npm run dev
```
